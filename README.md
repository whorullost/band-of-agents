# band-of-agents
For Lablab.ai's Band of Agents Hackathon
By Team [INSERT NAME HERE]! 
(whorullost & tropilemon)

Summary:
A multi-agent AI system with 6 specialized agents, including Onboarding, Security, Absences, Benefits, Payroll calculator, and Payroll Reporter, allowing for employees of a small to medium sized company to request information about payroll. There is anonymization of personal data with permission gateway, determining which AI's or users are allowed to access certain data. Returns paycheck reports with explanations and absence and benefit tracking. Using Band as a system to have agents communicate with each other in the chat room, allowing for any agent for being HR's entry point after being verified.
